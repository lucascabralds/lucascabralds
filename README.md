## 📦 Projeto em Destaque

### 🔗 [LinkMonitory — Sistema de Monitoramento e Auto-Cura de Links](https://github.com/lucascabralds/link-monitor-autocure)

Automação Python desenvolvida para lojistas afiliados do Mercado Livre que utilizam links encurtados em campanhas. O sistema monitora continuamente se os links de produto estão ativos via **Playwright (browser headless)** e, quando um produto sai do ar, executa a **auto-cura automática** — substituindo o link pelo próximo da fila de reservas e atualizando **Short.io** e **Supabase** em tempo real, sem nenhuma intervenção manual.

`Python` `Playwright` `Supabase` `Short.io` `Pandas` `OpenPyXL`
